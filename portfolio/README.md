# Shri Raam Balaji Portfolio

This is my personal portfolio website built using HTML and CSS. I created this project to showcase my skills, projects, and interests in web development, video editing, artificial intelligence, and embedded systems.

The website serves as a digital space where visitors can learn more about me, explore my work, and connect with me through my social platforms.

## About the Project

I wanted to create a portfolio that reflects both my technical skills and creativity. Instead of using a simple layout, I designed the website with a modern glassmorphism style, gradient effects, and smooth interactions to make it visually appealing.

The portfolio introduces who I am, highlights my skills, showcases some of my projects, and provides easy ways to get in touch. Throughout the project, I focused on creating a clean user experience while keeping the design responsive across different screen sizes.

## How to Run

Download or clone the repository.

Open the project folder.

Make sure the profile image is available in the project directory.

Open `index.html` in any modern web browser.

No additional installation or setup is required.
