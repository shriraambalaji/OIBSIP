Aurora Thermal Converter

A simple yet visually appealing Temperature Converter built using HTML, CSS, and JavaScript. This project allows users to convert temperature values between Celsius, Fahrenheit, and Kelvin while providing a smooth and interactive user experience.

About the Project

I created this project to practice JavaScript logic, form validation, and modern CSS design techniques. Instead of making a basic converter, I wanted something that looked attractive and felt responsive to user interactions.

The application features animated gradients, glowing background effects, dynamic temperature themes, and validation checks to ensure accurate conversions.

Features

 Convert temperatures between:

 Celsius (°C)
 Fahrenheit (°F)
 Kelvin (K)

 Clean and modern user interface


 Responsive design for different screen sizes

 How to Use

1. Enter a temperature value.
2. Select the unit you want to convert from.
3. Select the unit you want to convert to.
4. Click Convert Now or press Enter
5. The converted temperature will be displayed instantly.

Validation Included

The converter prevents impossible temperature values below Absolute Zero.

| Unit       | Minimum Allowed Value |
| ---------- | --------------------- |
| Celsius    | -273.15°C             |
| Fahrenheit | -459.67°F             |
| Kelvin     | 0 K                   |

If an invalid value is entered, the application displays an error message instead of performing the conversion.
