 Haven – Workspaces Built For Humans

A simple landing page for a modern coworking space, built using HTML, CSS, and JavaScript.

The goal of this project was to create a clean and professional-looking website while practicing responsive web design, animations, and JavaScript interactions. I focused on keeping the design minimal, easy to navigate, and visually appealing.

## About the Project

Most landing pages either look too crowded or too plain. For this project, I wanted to build something that feels modern and welcoming.

The website includes a hero section, feature cards, customer testimonials, and a footer section. I also added subtle animations and interactive elements to make the experience feel more engaging.

This project helped me improve my understanding of layout design, responsiveness, and DOM manipulation using JavaScript.

## Features

* Responsive landing page design
* Fixed navigation bar with scroll effect
* Hero section with call-to-action button
* Feature cards with hover animations
* Interactive testimonial slider
* Automatic testimonial rotation
* Floating information card animation
* Clean and modern UI
* Mobile-friendly layout

## Technologies Used
